# Voiceback Landing Page

A modern, multilingual landing page for Voiceback built with Next.js, TypeScript, and Tailwind CSS.

## 🌟 Features

- **Multi-language Support**: Spanish (default), English, and Brazilian Portuguese
- **Responsive Design**: Mobile-first approach with 880px breakpoint
- **Glassmorphism UI**: Beautiful glass cards with backdrop blur effects
- **Interactive Phone Mockup**: Realistic chat interface preview
- **Dark Mode**: Sleek dark theme with green accent colors

## 🛠️ Tech Stack

- **Framework**: Next.js 16.2.9 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Fonts**: Inter (Google Fonts)
- **Deployment**: Vercel

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/zakine/voiceback-smart-assistant.git
cd voiceback-smart-assistant
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) to view the landing page

## 🌍 Language Support

The landing page supports three languages that cycle in this order:
- 🇪🇸 Spanish (Español) - Default
- 🇺🇸 English 
- 🇧🇷 Portuguese (Português)

Language preference is automatically saved in localStorage.

## 📱 Responsive Breakpoints

- **Mobile**: < 880px (single column layout, hidden nav links)
- **Desktop**: ≥ 880px (full grid layout, visible navigation)

## 🎨 Design System

### Colors
- **Primary**: #25D366 (WhatsApp Green)
- **Backgrounds**: Dark charcoal with subtle green tints
- **Glass Effects**: Semi-transparent overlays with backdrop blur

### Typography
- **Font Family**: Inter
- **Weights**: 400, 500, 600, 700, 800

## 🚀 Deployment

This project is optimized for deployment on Vercel:

1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect Next.js and configure settings
3. Deploy with zero configuration needed
